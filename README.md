# 📒 Contact Book (Python CLI App)

This is a simple Python command-line Contact Book that allows users to add, search, and delete contacts. All data is stored in a `contacts.json` file.

## 🔧 Features
- Add a new contact
- Search existing contacts
- Delete a contact
- Simple file-based storage using JSON

## 🛠 Tech Stack
- Python
- File handling & JSON

## 🧠 What I Learned
- How to read/write JSON files
- Structuring a CLI app
- Modular function-based coding

```bash
python contact_book.py
